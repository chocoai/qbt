用户注册协议
球包通为向球包通会员所提供服务，特制定《球包通会员服务协议》（以下称为本协议）如下。

第一条 总则

  • 1.1 球包通的所有权和运营权归上海格灵信息科技有限公司所有。
  • 1.2 本协议用于规定球包通会员的资格、权利和义务。另有《球包通会员服务协议实施细则》（以下称为实施细则）规定具体实施办法及相关条款的解释。本协议和实施细则为一体，不得分别解释。

第二条 会员资格

  • 2.1 申请球包通会员者必须是球包通的用户。
  • 2.2 经过本人申请，认真阅读并接受本协议后方可申请注册为球包通会员。
  • 2.3 用户一经注册成为球包通会员，即被视为同意并接受本协议。会员按规定缴纳会费。已缴纳会费不论何种理由，均不退还。
  • 2.4 会员注册采用申告制。用户须承诺使用实名、正式住址以及所属手机号码登记注册信息。如用户登记虚假身份信息、或使用不完全、或过时信息注册，则须承担由此引起的所有后果以及相关责任（包括所有与此相关的法律责任），球包通将保留追究该用户法律责任的权利。

第三条 用户帐号

  • 3.1 经球包通网站（以下称为本站）本站规定的登记手续完成注册成为会员后，即获得本站发行的会员账号，享有会员的权限并承担本协议以及实施细则规定的义务。球包通有权对会员的权限设置进行变更。
  • 3.2 会员有义务妥善保管会员的账号和密码，并承担使用该账号和密码所进行的一切活动引起的后果的全部责任，本站不承担会员的行为所造成的后果的任何责任。
  • 3.3 如会员发现帐号遭到未授权的使用或发生其他任何安全问题，应立即修改密码并妥善保管，并及时通知本站。因黑客行为或用户的保管疏忽导致帐号非法使用，本站不承担任何责任。
  • 3.4 如因会员登记时提供的个人信息有任何错误、不实或不完整，或球包通有合理有合理的理由怀疑前述资料包含不实内容、或不完全时，球包通有权暂停或终止您的帐号，并拒绝您于现在和未来使用球包通全部或部分的服务。

第四条 服务

  • 4.1 球包通按照本协议以及实施细则向会员提供服务。
  • 4.2 球包通有权在必要时修改服务条款。用户应当及时关注并同意本站不承担通知义务。本站的通知、公告、声明或其它类似内容是本协议的一部分。
  • 4.3 会员可根据本协议以及实施细则的规定使用球包通提供的服务。
  • 4.4 会员账号只供本人使用，除有特殊说明时以外，享受服务的对象仅为会员本人。
  • 4.5 如会员在使用会员身份为自身以外的任何人享受面向会员的球包通服务，或者利用球包通业务赚取他人费用差价，或其他违反本协议以及实施细则规定的行为，或球包通有合理的理由怀疑前述事项为实，球包通有权暂停或终止该会员的帐号，并可拒绝其于现在和未来使用球包通全部或部分的服务。并保留保留追究相关法律责任的权力。
  • 4.6 除有特殊说明时以外，会员在享受会员服务待遇时，不得再同时享受球包通的其他优惠和减价服务。
  • 4.7 会员有义务接收球包通通过事先得到会员同意的方式提供的广告或其他商业信息。

第一条 适用范围

  • 1.1 本细则适用于本协议所描述的向球包通会员提供的服务的具体实施方式和标准。
  • 1.2 本细则将规定球包通会员的类别、注册方式、服务对象、服务内容、费用标准、禁止和处罚事项等。

第二条 会员类别

  • 2.1 球包通用户经过本人申请并按后述手续，即可成为会员。购买不同的会员卡，所享受的会员权益及义务也不相同。
  • 2.2 球包通会员卡共有2种不同卡种，分别为名士卡和畅享卡，用户可根据自身需求选择。
  • 2.3 畅享卡只有网络虚拟卡，不发放实体卡。
  • 2.4 会员可享受后述各项服务并承担由此产生的义务。

第三条 会员注册

  • 3.1 球包通用户在球包通网站或公众号上，以个人申告方式使用实名（本人的姓名、性别、身份证号码、住址、手机电话号码以及其他规定信息）登记申请、缴纳所定会费，并经球包通核准后，既可注册称为会员。
  • 3.2 会员须保证登记所使用个人信息完整真实、不含虚假信息、不含已过时信息。
  • 3.3 会员注册成功后，球包通将赋予其会员账号。
  • 3.4 会员卡在付款后可选择当即激活生效，也可以选择暂不激活，球包通会给予一个激活码，需要激活时致电400-609-0990进行开通。

第四条 会费

  • 4.1 球包通会员卡共有两种卡种，年费分别为：名士卡2000元，畅享卡3999元。
  • 4.2 下一年的年费可以提前缴纳。
  • 4.3 年费一旦缴纳，任何原因都概不退款。

第五条 服务对象和内容

  • 5.1 根据球包通的业务内容，规定服务对象为高尔夫球包。
  • 5.2 球包通所提供服务内容为根据会员要求运送高尔夫球包。

第六条 会员有效期

  • 6.1 球包通会员卡在激活后，有效期为1年。
  • 6.2 有效期满后，会员将自动转为普通会员，不再享受会员待遇。在期满前或期满后，都可以通过缴纳下一年年费的方式来续费，以继续享有原有权益。
  • 6.3 会员卡一经激活，权益期间用户不得以任何原因暂停会员卡服务，并往后顺延要求暂停时间作为补偿。
